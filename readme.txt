1. Clone the project to your pc
2. Navigate inside the mysite directory
3. Set up your python virtual environment for this project and activate it
4. Install the project dependencies using requirements.txt file by running following command
    --->$pip install -r requirements.txt 
5.Run the server by using following command in terminal
    --->$python manage.py runserver
    #default port is 8000

6.Now go to your browser and browse the urls as mentioned below:
    -->http://127.0.0.1:8000/polls/home

7.Navigate through hyperlinks to see the features listed.
    (a)first create user and use it to login to polls app
